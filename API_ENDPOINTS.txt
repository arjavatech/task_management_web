=== CURRENT API FIELD MAPPINGS ===

Company API Fields (POST /company):
- name → name
- EIN → ein (app uses 'ein')
- stateIncorporated → stateIncorporated
- contactPersonName → contactName (app uses 'contactName')
- contactPersonPhNumber → contactPhone (app uses 'contactPhone')
- address1, address2, city, state → same
- zip → zipCode (app uses 'zipCode')

Task API Fields:
- companyId → companyId
- name → name
- description → description
- dueDate → dueDate
- status → status
- completionDate → completionDate

Template API Fields:
- name → name
- description → description
- estimatedDays → estimatedDays

Template Assignment Fields:
- companyIds → companyIds (array)
- startDate → startDate
- dueDate → dueDate

User Profile Variables:
- email (string)
- name (string)
- phone (string)
- reminderSettings (object)
  - enabled (boolean)
  - daysBefore (number: 1,2,3,7,14)
  - timeOfDay (string: HH:MM)
- ccEmails (array of strings)

Modal State Variables:
- show (boolean)
- type (string: 'success', 'error')
- message (string)

Local State Variables:
- user (object)
- companies (array)
- tasks (array)
- taskTemplates (array)
- modal (object)
